# Tone and stance

Provenance: evidence-based disagreement order is partly `corpus-derived`; detailed language patterns are `conservative-implementation` and `synthetic-example`.

## Core posture

Lead with the answer. Use thanks selectively, acknowledge the scientific concern, and make evidence and action auditable. Do not treat politeness as a substitute for substance.

## Direct-answer patterns

- acceptance: `We agree that [specific issue]. We have [verified action] in [location].`
- clarification: `[Direct factual answer]. We revised [location] to make this distinction explicit.`
- partial agreement: `We agree that [supported portion]. The supplied evidence does not establish [remainder], so we [action].`
- missing input: `A final response requires [specific fact]. We therefore mark this item as AUTHOR_INPUT_NEEDED.`

## Disagreement

Use this order:

1. Restate the concern fairly.
2. Give the direct answer.
3. Cite a supplied artifact or scientific boundary.
4. Explain the manuscript clarification or narrowing.
5. State what remains unresolved.

Pattern: `We agree that [criterion] is important. Under [verified protocol], [supplied evidence]. This evidence supports [bounded claim] but not [stronger interpretation]. We have clarified this boundary in [location].`

## Reviewer misunderstanding

Treat it first as a manuscript-clarity failure:

Avoid: `The reviewer misunderstood our method.`

Prefer: `We agree that the original description did not make [distinction] sufficiently clear. We now state [verified clarification] in [location].`

Correct a factual error respectfully and precisely when the record is clear.

## Out-of-scope work

Do not cite time, cost, or convenience as the primary scientific reason. Explain the study-design dependency, provide available evidence, narrow the claim, and state the remaining limitation.

## Statistical concern

Name the experimental unit, test, assumptions, correction, effect/uncertainty, and location only when supplied. If any are missing, request them. Avoid `the result remains significant` without exact verified output.

## Citation request

Do not promise to cite an unverified paper. State that the citation will be evaluated for relevance; once verified, explain how it changes positioning rather than merely adding it to a list.

## Weak or forbidden patterns

- `The reviewer clearly misunderstood...`
- `Obviously`, `clearly`, `undoubtedly`, or `we strongly believe` without evidence.
- `We have revised accordingly` without action and location.
- `This is outside the scope` without explaining the claim boundary.
- `Due to time limitations` as the main refusal.
- `As requested, we added experiments` without supplied results.
- repeated gratitude formulas that consume a limited response budget.

## Claim-strength verbs

Match `establishes`, `supports`, `suggests`, `is consistent with`, and `does not establish` to supplied evidence. Preserve reviewer wording in the quoted comment even when the response uses narrower language.
