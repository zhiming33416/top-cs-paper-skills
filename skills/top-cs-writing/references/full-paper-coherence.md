# Full-paper coherence

Build an argument graph for full-manuscript work. Give each major claim a stable ID and compare its wording, evidence, and boundary across the title, abstract, introduction, experiments, discussion, limitations, and conclusion.

Report:

- claims that grow stronger across sections without new evidence;
- contribution items with no direct result, theorem, or analysis;
- results that are never used in the central argument;
- terminology that changes the apparent identity or scope of a component;
- decisive evidence placed only in an appendix or supplement that reviewers may ignore.

For a section-only request, create only the local graph edges that can be verified and mark external dependencies as `[MANUSCRIPT CONTEXT NEEDED]`.
