# Paper-type evidence overlays

Load the shared paper-type profile selected by the manifest, then apply these graph checks:

- **Empirical:** question -> protocol -> comparator -> uncertainty -> result -> boundary.
- **Algorithmic:** problem -> algorithmic idea -> assumptions/complexity -> matched-budget comparison -> sensitivity.
- **Theoretical:** definitions -> assumptions -> theorem -> proof intuition -> proof -> regime and comparison to prior bounds.
- **Systems:** requirement -> design decision -> workload -> operating point -> end-to-end result -> bottleneck/cost/reliability.
- **Dataset/benchmark:** task need -> construction -> coverage/exclusions -> quality/leakage -> evaluation -> maintenance/ethics.

These are contribution-type checks, not venue style rules. Mixed papers may need a primary overlay plus explicitly named secondary checks.
