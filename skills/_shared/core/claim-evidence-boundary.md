# Claim–evidence–boundary model

Build the paper around a small set of auditable claims.

For every major claim, record:

| Field | Required content |
|---|---|
| Claim | One falsifiable sentence, scoped to the studied setting. |
| Evidence | Figure, table, theorem, qualitative analysis, or external source that directly supports it. |
| Comparator | Baseline, prior work, null model, or expected behavior that makes the evidence meaningful. |
| Boundary | Dataset, task, population, assumptions, compute regime, or failure case limiting the claim. |
| Status | `supported`, `partial`, `missing`, or `contradicted`. |

Use this table before drafting. Do not let multiple sections make incompatible versions of the same claim. If evidence is partial, narrow the claim or request the missing test.
