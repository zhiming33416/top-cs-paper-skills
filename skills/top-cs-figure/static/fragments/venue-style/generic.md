# Venue Figure Style: Generic

Use when the venue is unknown, unsupported, or visual-style evidence is insufficient.

- Favor accessible, low-saturation palettes and ordinary matplotlib/PDF-safe typography.
- Use a single neutral family, one signal family, and one accent family per figure.
- Prefer simple evidence structure: one main comparison or mechanism panel plus subordinate checks.
- State that venue-specific visual style was not asserted.
