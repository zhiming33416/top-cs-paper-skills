# Venue Figure Style: ICML

Evidence status comes from `visual-style-rules.yaml`. Use a derived profile only when both `style_status: promoted` and `profile_status: usable`; otherwise retain generic semantic roles and report the evidence gap.

- Prefer clean quantitative panels with compact axes, small sans-serif text, and muted method colors.
- Use semantic method/baseline/ablation roles; assign hues only from usable derived evidence or the generic fallback palette.
- Keep legends short; use direct labels when a line or bar group remains spatially stable.
- At the 2026-07-13 snapshot 30 verified independent sources support usable anchors and profiles. Blue/red/green/magenta/violet are observed anchor families; pure corpus reds remain fill/accent evidence and are reconstructed for thin-line or text use. Re-read the live rules before every venue-specific claim.
- For camera-ready or format-sensitive work, verify the current official ICML style files and paper checker separately.
