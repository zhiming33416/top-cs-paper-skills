# Forest Interval

Use when estimate uncertainty is the primary evidence. Require label, estimate, lower, and upper columns.

- Order rows by declared domain order or an explicit sort transform.
- Draw a meaningful null/reference line supplied by the contract.
- Use marker shape as redundant method or subgroup identity.
- Never infer interval type from column names.
- Prefer horizontal intervals for readable labels.

Caption obligations define estimator, interval type, confidence/spread, sample unit, and null value. QA rejects reversed bounds, clipped intervals, unknown interval provenance, and overlapping row labels.
