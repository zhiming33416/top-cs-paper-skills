# Visual Family: Matrix And Heatmap

Use for task-by-method matrices, confusion matrices, attention or interaction maps, dataset coverage, correlation, and ablation grids.

- Choose a color map whose order matches the metric direction. Avoid rainbow maps.
- Fix row and column order by manuscript logic, clustering, or supplied ordering; report the choice.
- Put exact values in cells only when the matrix remains readable at final size.
- Normalize only with a documented denominator. Do not mix raw counts and percentages without labels.
- Add row/column grouping annotations when the claim depends on families of tasks, datasets, or models.
