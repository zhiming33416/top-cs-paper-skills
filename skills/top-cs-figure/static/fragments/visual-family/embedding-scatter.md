# Visual Family: Embedding And Scatter

Use for UMAP/t-SNE/PCA-style embeddings, calibration scatter, correlation, error analysis, and cluster diagnostics.

- Distinguish exploratory embedding from quantitative evidence. Do not treat visual separation as proof without a supplied metric.
- Label axes with the projection method or measured variables; avoid implying semantic axes for arbitrary embeddings.
- Use transparency, rasterized points, or density summaries for large point clouds while preserving legends and labels.
- Directly label stable clusters when possible; otherwise keep legends compact.
- For correlation claims, include supplied fit, interval, and statistic only when available.
