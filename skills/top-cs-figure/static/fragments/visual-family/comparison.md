# Visual Family: Comparison

Use for method-vs-baseline results, ablations, benchmark tables converted to figures, and subgroup comparisons.

- Prefer dot/range plots or grouped bars with intervals over plain bars when uncertainty matters.
- Keep metric direction visible: label `higher is better` or `lower is better` when the metric is not obvious.
- Order methods consistently across panels: proposed method first or last, then strongest baselines, then simple baselines. Explain any nonstandard order.
- Show per-dataset grouping only when it helps comparison; otherwise split panels or use small multiples.
- Do not imply statistical significance without supplied uncertainty, test, and comparison definition.
