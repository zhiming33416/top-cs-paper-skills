# Qualitative Image Plate

Use only with supplied images that constitute declared qualitative evidence. V3 requires image path, stable image ID, crop provenance, and aspect ratio.

- Preserve source aspect ratio and deterministic ordering.
- State crop/resizing operations; reject undeclared substitutions.
- Add a scale bar when physical distance or size is interpreted.
- Keep labels outside important image content.
- Do not use generated or reconstructed examples as experimental evidence.

Caption obligations map image IDs to cases, methods, and selection policy. QA checks existence, aspect ratio, provenance fields, scale-bar metadata when applicable, and nonblank output.
