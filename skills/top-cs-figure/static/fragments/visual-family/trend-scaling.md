# Visual Family: Trend And Scaling

Use for learning curves, data-scaling, model-size scaling, latency/throughput tradeoffs, time series, and sensitivity sweeps.

- Put the causal or controlled variable on the x-axis and state whether the axis is linear or log scaled.
- Use bands or error bars only when runs, folds, or intervals are supplied.
- Mark operating points such as budget, latency threshold, context length, or training-data size when they are part of the claim.
- Keep throughput, latency, memory, and accuracy tradeoffs on separate axes unless the relationship is central and legible.
- Avoid smoothing that hides supplied observations; if smoothing is used, preserve raw points.
