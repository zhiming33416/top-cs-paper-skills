# Layout Archetypes

Choose layout from the claim and evidence chain.

## Quantitative grid

Use for benchmark comparisons, ablations, robustness, and sensitivity sweeps. Keep shared axes and panel labels consistent.

## Schematic-led composite

Use when the paper needs a method or system map plus evidence panels. The schematic must encode real data/control flow and use manuscript terminology.

## Image-heavy

Use only when image panels are primary evidence. Keep dark backgrounds only for image plates where the source images need it.

## Sparse theoretical

Use for proof, algorithm, or conceptual figures with few quantitative marks. Avoid decorative density.

## Mixed

Use one hero panel plus subordinate checks when the figure must combine mechanism, result, and failure analysis. Do not make every panel equal if one panel carries the main claim.
