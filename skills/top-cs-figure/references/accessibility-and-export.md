# Accessibility and Export

Check publication figures in color and grayscale. Decision-critical categories need either luminance separation, labels, markers, or line styles in addition to hue.

- Keep SVG text editable and use a stable sans-serif font stack.
- Export PDF and 300-DPI PNG previews with the SVG; require TIFF only when the submission target requires it.
- Check raster previews for blank output, low DPI, clipped content, and unexpected edge contact.
- Use explicit metric-direction text, readable axis labels, and panel labels that agree with the caption.
