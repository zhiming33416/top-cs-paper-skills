# Venue-Specific Figure QA

Use this before final delivery when a venue is named.

- Verify official format, page, template, and camera-ready rules outside corpus-derived visual evidence.
- If `visual-style-rules.yaml` marks the venue as `preliminary`, say the style is preliminary and conservative.
- If it marks the venue as `external-corpus-required`, do not assert learned venue style.
- Check that figure size, line weights, font sizes, and color contrast remain readable after placement in the target template.
- For WWW Research, ensure the figure caption or manuscript callout preserves the Web relevance when the figure supports a Web-specific claim.
